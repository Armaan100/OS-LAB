# Map
- it is an associative container that stores key-value pairs in a sorted order by key
- it does not allow duplicate keys

## Library
- <map>

## Constructors
- map()
- mao(initializer_list)

## Capacity FUnctions
- size()
- empty()

## Modifier
- insert(pair)
- erase(key)
- clear()

## Lookup FUnctions
- find(key)
- count(key)
- at(key)
- opeartor[]
